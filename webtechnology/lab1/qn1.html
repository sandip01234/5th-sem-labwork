HTML Basics

Use of <p> Tag

The <p> tag is used to define paragraphs in HTML. It automatically adds spacing before and after the text to make it more readable.

Use of <pre> Tag

        The <pre> tag is used to 
        display preformatted text in HTML. 
                             It preserves both spaces and  
        line breaks, making it useful       for displaying code or text as-is.
